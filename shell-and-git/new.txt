1. open terminal and git hub

2. create a new repository on GitHub.com

3. navigate to the folder with cd on the terminal

4. check the status with git status command

5. use git add command 
To stage a specific file:
git add filename.txt

To stage all changes:
git add .

6. commit the changes with a message 
git commit -m "Your commit message here"


7. push changes to git hub: git push origin main
